Twitch Auto Refresh Plus

About

Twitch Auto Refresh Plus is a Chrome extension designed to automatically refresh Twitch.tv livestreams when they encounter connection issues, bandwidth problems, or inactivity. It helps maintain uninterrupted viewing by detecting stream interruptions and automatically attempting to restore the connection.

Features

  • Automatic detection of stream interruptions
  • Configurable refresh intervals (5-60 seconds)
  • Adjustable retry attempts (1-20 times)
  • Visual status indicator
  • Stream status messages
  • Works only on active livestreams
  • Smart detection of permanent stream end

How It Works

When you load a Twitch.tv channel that's currently streaming:

  • The extension activates automatically if enabled
  • A small "T" indicator appears to show it's active
  • If the stream encounters issues, automatic refresh begins
  • The extension will retry based on your settings
  • Status messages keep you informed of the progress
  • Stops automatically if the stream truly ends

Settings

  • Enable Auto Refresh: Turn the feature on/off
  • Refresh Interval: Time between refresh attempts (5-60 seconds)
  • Max Retry Attempts: Number of refresh attempts before stopping (1-20)
  • Show Status Messages: Display refresh countdown and status
  • Show Overlay Icon: Show/hide the "T" indicator
